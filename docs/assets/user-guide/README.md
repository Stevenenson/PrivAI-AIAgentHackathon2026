# Privai User Guide Screenshots

The printable booklet at `docs/privai_user_guide_booklet.html` expects these
screenshots in this folder:

- `01-starting-page.png`
- `02-settings-connect-board.png`
- `03-device-model-picker.png`
- `04-how-it-works.png`
- `05-new-chat-composer.png`

If the images are missing, the booklet still opens and shows descriptive
fallback figure cards.
